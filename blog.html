Frequently Asked Questions?

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a webpage as a tree of objects, allowing manipulation of HTML and CSS.

2. How do you select an element from the DOM?

You can select an element from the DOM using its ID, class name, tag name, or other criteria. Here are some common methods: document.getElementById(), document.querySelector(), or document.getElementsByClassName().

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in which you attach an event listener to a parent element instead of directly to the child elements. This allows you to handle events for multiple child elements efficiently. It's useful for dynamic content or when you have many elements with the same event handler.

4. How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, use setAttribute() and getAttribute(). To manipulate styles, use style.propertyName or classList for class-based styling.
